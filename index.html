PT-P2 kNN Predictive Modelling

Description

The activity demonstrates the use of the k-Nearest Neighbors (kNN) algorithm to perform predictive analytics using the Iris dataset. The project involves loading the dataset, splitting it into training and testing data, building the model, generating predictions, evaluating performance, and visualizing the results to analyze how parameters affect classification accuracy.

Predictive Skills

The activity uses Python libraries such as sklearn datasets and Matplotlib for machine learning and visualization. Methods like train_test_split() dividing the dataset into 80% training data and 20% testing data, allowing the model to be evaluated on unseen data also the kNN value was set to 5 for the model to consider 5 nearest data points when making a decision. The fit() method trains the model, while predict() generates predictions. Evaluation tools such as accuracy_score(), classification_report(), and confusion_matrix() are used to measure the model’s performance and analyze prediction results.

The “So What”

Learning predictive analytics and machine learning techniques is valuable because many industries now rely on data-driven decision making. Companies now look for people who have skills such as building predictive models, analyzing datasets, and evaluating model performance as it is now becoming a trend in the industry.

By understanding algorithms like kNN and using Python tools for data analysis, I can develop skills that will help me stay competitive and serve as my leverage to others in the industry.

Reflection

One of the challenges I have faced during this activity was that it was my first time using Python Notebook as well as predictive analytics and machine learning algorithms. Because of this, I find it difficult to understand how the environment works and how the code functions together to produce results. I overcame this challenge by familiarizing myself with the notebook environment and carefully studying how each part of the program function to better understand how the program works through the phases such as cleaning, split and testing, and evaluating the model.

Predictive Analytics Article

Description

Predictive analytics is a method that makes use of statistical algorithms, machine learning techniques, and historical data to find trends and predict future results. Predictive analytics is used by businesses in today’s industries to better understand relationships, cause and effect, and trends. Businesses can use predictive analytics to turn raw information into useful insights that enhance operational effectiveness and strategic planning as they produce more and more data.

The “So What”

Gaining knowledge of predictive analytics is crucial because it enables professionals to evaluate data and make reasonable choices. Predictive analytics helps businesses anticipate risks, increase forecasting accuracy, and maximize performance in real-world sectors like healthcare, supply chains, and business operations. For instance, supply chains can anticipate disruptions before they happen, hospitals can detect early disease patterns, and businesses can forecast consumer demand. These skills increase productivity, cut expenses, and support businesses in maintaining their competitiveness in a market that is changing quickly.

Technical Challenges I Have Faced

One of the technical difficulties I encountered while researching predictive analytics was locating credible and reliable sources that explained its useful applications and advantages. Fewer online resources offer in-depth research or statistical proof demonstrating how predictive models enhance forecasting accuracy and decision-making, despite the fact that many discuss predictive analytics in general. As a result, I had to thoroughly assess studies to make sure the data I used was accurate and relevant to the subject.
